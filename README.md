publishing-automation
========================================

Publish markdown files to PDF

Requirements & Setup
----------------------------------------

### Requirements

- Node.js v18+ 


### Setup

```bash
npm install
```


#### Additional step for WSL

```bash
sudo apt install libgtk-3-dev libnotify-dev libgconf-2-4 libnss3 libxss1 libasound2
```


How to Build
----------------------------------------

### Demo

```bash
npm run build-demo
```
